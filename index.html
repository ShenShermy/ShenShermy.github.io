Portfolio.

Individual Research · Featured
HateFusion-Light — Multimodal Hate Speech Detection
2024 – 2025
Problem

Detecting hate speech in memes requires understanding both image and text together — neither modality alone is sufficient. Existing solutions are either too large for edge deployment or ignore multimodal context entirely.

Data

Facebook Hateful Memes dataset: 10,000+ image–text pairs labelled for hate speech, covering sarcasm, derogatory stereotypes, and implicit bias.

Approach

Fused ModernBERT + HateBERT (text) and SigLIP-2 ViT-B/16 (image) via a four-stage pipeline: modality quality assessment → bidirectional cross-modal attention → gated dynamic fusion with anti-collapse regularisation → supervised contrastive learning. Only 1.5M trainable parameters via LoRA. Applied Adaptive Focal Loss with multi-task learning (sentiment + sarcasm).

Outcome / Impact

Expected AUROC 87–89% and F1 83–85% on a consumer-grade GPU (6–8 GB VRAM). Delivered end-to-end pipeline with Gradio demo and production-ready FastAPI service — deployable without data-centre hardware.

Python PyTorch HuggingFace LoRA ModernBERT SigLIP-2 FastAPI Gradio ↗ GitHub

Fake News Detection — Multi-Model NLP Comparison
2024
Problem

Misinformation spreads rapidly online. The challenge was to identify which NLP architecture most reliably classifies news articles as real or fake.

Data

Public fake-news corpus with thousands of labelled news articles; text preprocessed with tokenisation and contextual embeddings.

Approach & Outcome

Implemented and compared BiLSTM, fine-tuned BERT, and a hybrid BERT-BiLSTM. Built an automated evaluation pipeline generating five comparative visualisations that auto-highlight the best-performing model per metric.
